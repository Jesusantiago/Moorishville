NOTICE AND DISCREIMER

Welcome to the new and developing remote file access site, Moorishville County Divine Register of Deeds, an unincorporated non-profit religious county under article 508(c)(1)(a). This site contains copies of records and/or unofficial records provided as a public service.

The official records’ location is currently pending via an in the common law legal notice, as the public area to hold these documents is still under development. Please contact EMPRESS RAYONNA EUKEYA EL for more information regarding the original documents’ location.

To use this site, you must have a computer that can display windows at a resolution of at least 1024 pixels in length. This site has been optimized for use in Google Chrome, and will display results using the Portable Document Format (.pdf). The user is advised to search with the full first and last names of the parties involved in whichever document being searched for in order to locate the desired result(s). If the document being searched for has been recorded recently, please allow time for the online record to be copied and displayed on this site. The online record copies will not be as up to date as the physical records.
